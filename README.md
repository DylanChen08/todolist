# todolist

> 根据慕课网教程改编的todolist

## Build Setup
 node_modules等等文件夹里面东西太多，在上传git之前已经在.gitignore里忽略掉了 ，需要安装依赖：
``` bash
npm install
```

然后就是跑起来啦！

``` bash

npm run dev

```
